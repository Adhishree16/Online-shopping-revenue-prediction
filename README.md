# Online-shopping-revenue-prediction
IBM SPSS Modeler project for predicting online shopping revenue using CHAID and C5.0 classification models on customer browsing behavior data
# Online Shopping Revenue Prediction Using IBM SPSS Modeler

## Project Overview
This project predicts whether an online shopper will generate revenue based on browsing behavior using IBM SPSS Modeler. Machine learning classification algorithms such as CHAID and C5.0 were used to analyze customer behavior and predict purchasing decisions.

## Objectives
- Predict customer purchase behavior
- Analyze online shopping patterns
- Compare machine learning models
- Improve business decision-making

## Tools Used
- IBM SPSS Modeler
- Online Shoppers Intention Dataset
- CHAID Algorithm
- C5.0 Algorithm

## Methodology
1. Data Collection
2. Data Preparation
3. Feature Selection
4. Data Partitioning
5. Model Building
6. Model Evaluation
